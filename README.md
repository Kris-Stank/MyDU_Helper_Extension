# MyDU Manager Helper — MVP

Локальное расширение Chrome/Edge для первичной проверки заявлений в MyDU.

## Что делает

- Показывает перемещаемую светлую панель в фирменном стиле AITU с отдельными экранами «Проверка», «Шаблоны» и «Комментарий».
- Сворачивается в перемещаемую кнопку с маскотом и сохраняет выбранную позицию.
- Делит шаблоны замечаний на три раздела карточки MyDU и содержит все 51 русских комментария из исходного HTML-хелпера без изменения текста.
- Находит только безопасные формальные признаки: возможную путаницу серии и номера, `Г.А.` в КАТО, адрес из цифр/дефисов и слишком короткое место работы родителя.
- Проверяет структуру адреса и показывает не более одного краткого предупреждения с действиями «Добавить» и «Игнорировать».
- Считает средний балл по числовым оценкам 5, 4, 3 и 2 до двух знаков и сравнивает его с MyDU.
- При открытии PDF обнаруживает удостоверение личности, свидетельство о рождении, аттестат/диплом, приложение с оценками или IELTS.
- Показывает поверх окна документа закреплённую карточку с относящимися к нему значениями из MyDU, чтобы менеджеру не приходилось закрывать PDF для сверки.
- Для PDF с доступным текстовым слоем показывает автоматические совпадения. Для сканов оставляет удобную ручную сверку по карточке MyDU.
- Собирает и копирует комментарий; решение и отправка всегда остаются за менеджером.
- Восстанавливает комментарий в течение 3 минут после перезагрузки или закрытия вкладки.

## Чего не делает

- Не нажимает «Отправить на доработку» и «Отправить тех.секретарю».
- Не загружает и не передаёт данные или документы во внешние сервисы.
- Не хранит поля заявления и документы: только текст черновика и выбранные замечания с ограничением в 3 минуты.
- Не запускает OCR и не обрабатывает изображения документов.
