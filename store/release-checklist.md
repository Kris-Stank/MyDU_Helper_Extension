# Чек-лист публикации версии 1.4.0

1. Открыть профиль издателя и подтвердить контактный email.
2. Убедиться, что в Google-аккаунте включена двухэтапная аутентификация.
3. Опубликовать `privacy-policy.html` на публичном HTTPS-сайте.
4. Нажать «Добавить продукт» в Chrome Web Store Developer Dashboard.
5. Загрузить `release/MyDU-Manager-Helper-1.4.0.zip`.
6. Вставить тексты из `dashboard-fields-ru.md`.
7. Загрузить файлы из `store/assets` и `store/screenshots`.
8. Выбрать видимость Unlisted или Private.
9. Добавить временный тестовый аккаунт MyDU и тестовое заявление с вымышленными данными.
10. Проверить, что в карточке указан публичный URL политики конфиденциальности.
11. Нажать «Отправить на проверку» и при необходимости включить отложенную публикацию.

Не публиковать реальные логины, пароли или персональные данные в описании и скриншотах. Тестовые учётные данные вводятся только в закрытый раздел инструкций для команды проверки Chrome Web Store.
