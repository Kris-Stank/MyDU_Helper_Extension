# Chrome Web Store — готовые поля для MyDU Manager Helper

## Основная информация

**Название**

MyDU Manager Helper

**Краткое описание из manifest.json**

Локальный помощник менеджера приёмной комиссии AITU. Не отправляет заявления самостоятельно.

**Язык**

Русский

**Категория**

Работа и планирование. Если в списке используется старое название категории, выбрать «Производительность».

**Подробное описание**

MyDU Manager Helper помогает уполномоченным сотрудникам приёмной комиссии проверять заявления абитуриентов в MyDU и подготавливать понятные комментарии для исправления ошибок.

Основные возможности:

- автоматическая проверка заполненных полей заявления;
- сравнение данных заявления с удостоверением личности и свидетельством о рождении;
- проверка сертификатов ЕНТ и IELTS;
- локальное распознавание текста документов на казахском, русском и английском языках;
- готовые шаблоны замечаний на русском, казахском и английском языках;
- добавление найденного замечания в комментарий одним нажатием;
- проверка наличия сведений хотя бы об одном родителе или законном представителе;
- калькулятор GPA и подготовка итогового комментария для копирования.

Все проверки и OCR выполняются локально на устройстве пользователя. Расширение не передаёт заявления, документы или распознанный текст разработчику либо сторонним сервисам. Расширение не принимает решение о поступлении, не изменяет статус заявления и не отправляет его автоматически. Итоговое решение всегда остаётся за сотрудником.

Расширение работает только на страницах заявлений MyDU по адресу `https://my-du.astanait.edu.kz/admission/applicants/*`.

## Единая цель расширения

Локальная помощь уполномоченным сотрудникам приёмной комиссии при проверке данных и документов абитуриентов в MyDU и подготовке комментариев по найденным несоответствиям.

## Обоснование разрешений

### storage

Используется только для локального хранения положения панели, выбранных настроек интерфейса и временного черновика комментария. Данные не синхронизируются и не передаются разработчику или третьим лицам.

### clipboardWrite

Используется для копирования подготовленного комментария в буфер обмена только после явного нажатия пользователем кнопки копирования.

### Доступ к хосту `https://my-du.astanait.edu.kz/*`

Необходим для отображения панели на страницах заявлений MyDU и локального чтения видимых полей заявления и открытых документов с целью их проверки и сравнения. Доступ ограничен одним доменом и заявленной функцией расширения.

## Удалённый код

**Ответ:** Нет, расширение не использует удалённый код.

Весь исполняемый JavaScript, OCR-воркер, OCR-ядро и языковые модели включены в пакет расширения. Документы не отправляются во внешние OCR-сервисы.

## Раскрытие обработки данных

Следует отметить:

- Personally identifiable information / Личная идентификационная информация;
- Location / Адрес или местоположение — поскольку в заявлении обрабатывается адрес проживания;
- Website content / Содержимое веб-сайта — поля заявления и открытые документы.

Не отмечать данные аутентификации, историю браузера, финансовую или медицинскую информацию: расширение их не запрашивает и не использует.

Для всех отмеченных категорий назначение — **App functionality / Функциональность приложения**.

Подтвердить, что данные:

- не продаются третьим лицам;
- не используются и не передаются для целей, не связанных с единой целью расширения;
- не используются для кредитования или определения кредитоспособности;
- не используются для персонализированной рекламы;
- не передаются разработчику или сторонним сервисам.

**URL политики конфиденциальности:** после размещения файла `privacy-policy.html` на публичном HTTPS-сайте вставить его адрес в это поле.

## Инструкции для проверки

Расширение работает только после авторизации в закрытой системе MyDU. Для проверки необходимо предоставить Chrome Web Store временный тестовый аккаунт без реальных персональных данных и тестовое заявление с вымышленными сведениями и документами.

1. Авторизуйтесь по адресу `https://my-du.astanait.edu.kz/` с предоставленными тестовыми данными.
2. Откройте тестовое заявление по адресу, соответствующему шаблону `/admission/applicants/{id}`.
3. Нажмите «Проверить поля» в панели MyDU Manager Helper.
4. Убедитесь, что расширение показывает найденные формальные несоответствия и позволяет добавить замечание.
5. Откройте тестовый PDF-документ в заявлении. Расширение покажет локально распознанные значения и результаты сравнения.
6. Перейдите в «Шаблоны», переключите язык комментариев и добавьте шаблон.
7. Перейдите в «Комментарий» и нажмите кнопку копирования.

**Ссылка на тестовое заявление:** `[вставить URL]`
