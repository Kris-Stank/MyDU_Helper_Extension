# Политика конфиденциальности MyDU Manager Helper

Дата вступления в силу: 22 июля 2026 года  
Версия расширения: 1.4.0

## 1. Назначение расширения

MyDU Manager Helper предназначен для локальной помощи уполномоченным сотрудникам приёмной комиссии при проверке данных и документов абитуриентов в системе MyDU и подготовке комментариев по найденным несоответствиям.

Расширение не принимает решения о поступлении, не изменяет статус заявления и не отправляет заявление автоматически. Окончательное решение и любые действия в MyDU выполняет пользователь.

## 2. Какие данные обрабатываются

При открытии страницы заявления расширение может локально обрабатывать отображаемые в MyDU сведения, необходимые для проверки, включая:

- фамилию, имя и отчество;
- дату рождения, ИИН и номера удостоверяющих документов;
- гражданство, адрес и контактные сведения;
- сведения о родителях или законных представителях;
- сведения об образовании, результатах ЕНТ и языковых сертификатах;
- содержимое прикреплённых документов и результаты локального распознавания текста;
- текст замечаний, выбранных или введённых пользователем.

Расширение не запрашивает пароли, платёжные данные, историю браузера или медицинскую информацию.

## 3. Как используются данные

Данные используются исключительно для заявленной функции расширения:

- сравнения сведений заявления с открытым документом;
- выявления возможных формальных несоответствий;
- локального OCR-распознавания документов;
- формирования комментария для последующего копирования пользователем.

Результаты автоматической проверки являются подсказкой и требуют визуальной проверки сотрудником.

## 4. Локальная обработка и хранение

Поля заявления, изображения документов и распознанный текст обрабатываются локально на устройстве пользователя и не сохраняются разработчиком.

OCR выполняется в памяти текущей вкладки с использованием файлов, включённых в пакет расширения. Документы и распознанный текст не отправляются во внешние OCR-сервисы.

Расширение использует локальное хранилище Chrome для положения панели и временного черновика комментария. Черновик становится недоступным через три минуты; просроченная запись игнорируется и удаляется при следующем открытии соответствующего заявления. Настройки положения панели хранятся до удаления расширения или очистки его данных.

## 5. Передача третьим лицам

Расширение не передаёт разработчику или третьим лицам данные заявлений, документы, распознанный текст, историю использования или содержимое комментариев.

Данные не продаются, не используются для рекламы, профилирования, кредитования или определения кредитоспособности.

## 6. Разрешения Chrome

- `storage` — локальное хранение положения панели, настроек интерфейса и временного черновика комментария;
- `clipboardWrite` — копирование подготовленного комментария только по явному действию пользователя;
- доступ к `https://my-du.astanait.edu.kz/*` — отображение панели и локальная проверка данных только на страницах MyDU.

## 7. Защита данных и Limited Use

Использование информации расширением соответствует правилам Chrome Web Store User Data Policy, включая требования Limited Use. Данные используются только для основной пользовательской функции, не передаются для рекламы и не предоставляются людям со стороны разработчика для чтения.

## 8. Управление данными

Пользователь может удалить локальные данные расширения, удалив расширение в Chrome или очистив данные расширения. Удаление расширения прекращает всю обработку данных.

## 9. Изменения политики

При изменении способов обработки данных эта политика будет обновлена одновременно с соответствующим обновлением расширения. Дата вступления в силу будет изменена в верхней части документа.
